Os atributos SRC (source), WIDTH, HEIGHT

src é o caminho da imagem. width é largura e height é altura. E alt é leitura alternativa, ou seja, quando a imagem não carrega ou quando o usuário possui problemas de visão.

O atributo STYLE

especifica o estilo do elemento como cor, fonte, tamanho, entre outros.

Exemplo: eu sou um parágrafo

O atributo LANG

Serve para indicar a linguagem padrão do documento. Muito importante para leitores de tela e mecanismos de busca. É declarado na tag inicial HTML.

O atributo título

Serve para atribuir um título a uma parte específica do conteúdo. Quando se passar o mouse em cima, mostrará este título. É estilo uma dica...
